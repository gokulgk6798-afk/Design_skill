# Brand Knowledge Architecture

## Purpose

Define the structure for future AI-readable brand knowledge.

## Audience

- Brand Owner
- Marketing team
- Content team
- Designers
- AI assistants

## Scope

This directory contains approved and source-derived brand knowledge modules for mission, vision, values, culture, brand personality, and company profile context.

## Modules

- `mission.md`
- `vision.md`
- `values.md`
- `culture.md`
- `brand-personality.md`
- `company-profile.md`

## Required Contents Per Module

- metadata
- canonical statement or rule
- explanation
- usage guidance
- AI usage notes
- dependencies
- examples placeholder
- anti-pattern placeholder

## Dependencies

- `docs/documentation/brand.md`
- `docs/documentation/writing-guidelines.md`
- `knowledge/_meta/metadata-schema.md`

## Owner

Brand Owner.
