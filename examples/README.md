# Examples

This folder contains approved reference examples that demonstrate how to apply the Organization Design Kit.

Examples are not templates. Templates define reusable structure. Examples show how an artifact should look or read when the Design Kit is applied correctly.

## Available Examples

- `presentation/company-profile-example.md`
- `proposal/proposal-example.md`
- `prd/prd-example.md`
- `ui-design/ui-design-example.md`
- `documentation/documentation-example.md`

## Usage

Use examples to understand:

- expected structure
- approved terminology
- required metadata
- validation checklist usage
- source files that should be loaded before generation

Examples must remain aligned with the current knowledge base and checklists.
