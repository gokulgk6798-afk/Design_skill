# Example Proposal: AI Transformation Proposal

## Purpose

Demonstrates the expected structure for a Finstein proposal.

## Required Sources

- `knowledge/brand/`
- `knowledge/documentation-standards/documentation-guidelines.md`
- `knowledge/ux/ux-writing.md`
- `templates/proposals/`
- `knowledge/checklists/documentation-checklist.md`

## Proposal Structure

1. Title
2. Purpose
3. Executive summary
4. Client context
5. Problem statement
6. Proposed solution
7. Scope
8. Deliverables
9. Timeline
10. Assumptions
11. Dependencies
12. Success metrics
13. Commercial summary
14. Approval
15. Version history

## Example Tone

Use clear, professional, confident, and human-centered language.

## Validation

Use `knowledge/checklists/documentation-checklist.md`.
