# Logo Assets

This folder contains approved logo source assets for the Organization Design Kit.

## Approved Assets

| Asset | File | Notes |
| --- | --- | --- |
| Finstein white logo | `finstein-logo-white.pdf` | Source PDF provided for Milestone 8. |
| Finstein dark logo | `finstein-logo-dark.pdf` | Source PDF provided for Milestone 8. |

## Usage

Use these assets only according to approved guidance in `knowledge/visual-identity/logos.md` and `knowledge/visual-identity/logo-usage.md`.

## Presentation Usage

Presentation generators must use:

- `finstein-logo-dark.pdf` on light or white backgrounds.
- `finstein-logo-white.pdf` on dark, deep-blue, or image backgrounds.

Do not omit the logo from cover slides, closing slides, or branded section dividers unless the requested output is only a text outline.

## Machine-Readable Manifest

Use `logo-assets.json` for exact asset paths and presentation usage rules.

## Format Limitation

PDF is the currently approved source format. If a presentation generation tool cannot embed PDF logos, it must request or create an approved PNG or SVG export before finalizing the deck. It must not replace the logo with text, an unapproved image, or a generated approximation.
