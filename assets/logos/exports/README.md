# Logo Exports

This folder is reserved for approved presentation-ready logo exports such as PNG or SVG.

Current approved source assets are PDF files:

- `assets/logos/finstein-logo-dark.pdf`
- `assets/logos/finstein-logo-white.pdf`

If a presentation generation workflow cannot embed PDF logo assets, create or request approved PNG or SVG exports and place them in this folder. Until exports are approved, assistants must not substitute generated or unofficial logo artwork.
