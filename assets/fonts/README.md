# Font Assets

This folder contains approved font source assets for the Organization Design Kit.

## Approved Font Families

| Font | Folder | License |
| --- | --- | --- |
| Roboto | `roboto/` | `roboto/OFL.txt` |

## Usage

Roboto is the approved font for all official and other presentations.

Use typography guidance in:

- `knowledge/design/typography.md`
- `tokens/source/typography.json`
- `tokens/source/font-sizes.json`
- `tokens/source/font-weights.json`
- `tokens/source/line-heights.json`

## Fallback Stack

Recommended fallback stack:

```css
font-family: "Roboto", Arial, sans-serif;
```
