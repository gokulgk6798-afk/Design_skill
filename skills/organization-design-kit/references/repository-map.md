# Repository Map

## Root Files

- `AGENTS.md`: AI assistant operating instructions.
- `README.md`: repository overview and quick start.
- `ROADMAP.md`: milestone status.
- `CONTRIBUTING.md`: contribution and review rules.
- `CHANGELOG.md`: change history.
- `LICENSE`: license recommendation.

## Knowledge

- `knowledge/brand/`: mission, vision, values, culture, brand personality, and company profile.
- `knowledge/visual-identity/`: logo assets and logo usage.
- `knowledge/design/`: colors, typography, spacing, grid, radius, and elevation.
- `knowledge/components/`: UI component standards.
- `knowledge/ux/`: UX writing, content, accessibility, and color accessibility.
- `knowledge/presentation-standards/`: presentation rules and company profile deck guidance.
- `knowledge/documentation-standards/`: documentation and email signature standards.
- `knowledge/patterns/`: design patterns, best practices, and anti-patterns.
- `knowledge/checklists/`: validation checklists.

## Tokens

- `tokens/source/`: platform-independent JSON design token sources.
- `tokens/platforms/`: future platform output targets.

## Templates

- `templates/presentations/`
- `templates/documents/`
- `templates/proposals/`
- `templates/workflows/`

## Prompts

- `prompts/_meta/prompt-standard.md`
- task-specific folders under `prompts/`

## Examples

- `examples/presentation/`
- `examples/proposal/`
- `examples/prd/`
- `examples/ui-design/`
- `examples/documentation/`

## Validators

- `validators/`: checklist entry points.
- `knowledge/checklists/`: review checklist content.
