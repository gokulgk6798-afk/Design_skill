the size of the symbol and ensures that the logo is not

on all sides of the logo.

tagline. It should be used in most instances. In cases where there

There are several approved versions of the Finstein 
brandmark. These versions have been created for the 
speci/f_ic situations described on this page.
Outline Version
The outline version should be used in special situations when 
the recommended solid brandmark versions (primary, second-
ary and reverse) do not work. These situations include engrav-
ing, embossing, etching, embroidery or any other situation 
where production values severely limit a clean reproduction of 
the non-outline versions. The outline version may be used in 
black or white only.

Georgia, Circular Std and Arial are the recommended Goldman 
Sachs fonts. Circular Std was chosen because it is clean and 
contemporary — setting the tone for the Finstien brand.  Georgia 
is a slightly more traditional typeface. Therefore, it provides a nod 
to the tradition and history of Finstien. Arial is used by the UI 
Toolkit for all digital apps. For PowerPoint and electronic media, 
Arial and Times New Roman should be used.
Georgia is a serif-style typeface, serifs being semistructural details on the ends of some of the 
strokes that make up letters and symbols within this family

Inline subhead SMALLCAPS INLINE SUBHEAD
The largest headline size used within a piece should be set in bold
Deep Blue unless white is needed for a dark background. Lower-level 
headlines and subheads may be set in black, dark blue or an accent 
colour. Body copy should be set in black. Warm gery should never be 
used for copy. See Colour section of the guideline for more information 
about colour usage.
Not all headline levels needs to be used in a single document/piece. 
Use a number of hierarchy levels that feel proportional and appropriate. 
In bulleted lists, letters and roman numerals should be used as needed.
