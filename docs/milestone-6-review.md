# Milestone 6 Review

## Scope

Milestone 6 creates the reusable AI prompt library.

## Completed Deliverables

- Prompt library index
- Prompt standard
- Prompt loading and selection guidance
- Presentation creation prompt
- Proposal creation prompt
- PRD generation prompt
- UI design prompt
- UX review prompt
- Research prompt
- Documentation prompt
- Accessibility review prompt
- Design review prompt
- User flow prompt
- Wireframe prompt
- Component creation prompt

## Out of Scope

- Assistant-specific integration files
- Claude-specific instructions
- Codex-specific instructions
- Organization-specific brand values
- Filled examples
- Automated validation
- Generated artifacts

## Review Questions

- Are all required prompt categories represented?
- Does every prompt include purpose, inputs, expected outputs, constraints, validation rules, examples, and version?
- Are prompt dependencies clear?
- Are prompts sufficiently vendor-neutral?
- Is assistant-specific behavior properly deferred to Milestone 7?

## Approval Gate

Milestone 7 should not begin until Milestone 6 is approved.
