# Presentations Documentation Specification

## Purpose

Define the required structure for future presentation documentation, including deck types, slide structure, layout rules, branding, charts, and accessibility.

## Audience

- Content Owner
- Brand Owner
- Sales
- Marketing
- Leadership
- AI assistants

## Scope

Includes the documentation contract for presentation standards. Does not include actual presentation templates or slide content.

## Contents

- presentation categories
- deck structure
- slide anatomy
- layout rules
- visual hierarchy guidance
- chart and data display guidance
- brand usage references
- accessibility expectations
- example references

## Dependencies

- `templates/` for future presentation templates
- `examples/` for future approved presentation examples
- `docs/documentation/brand.md`
- `docs/documentation/colors.md`
- `docs/documentation/typography.md`
- `docs/documentation/accessibility.md`

## Update Frequency

On template change or brand change.

## Owner

Content Owner.
