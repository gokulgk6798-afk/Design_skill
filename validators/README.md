# Validators

This directory contains review checklist entry points for Milestone 9.

These are markdown validators, not automated scripts. They are intended for human reviewers and AI assistants.

## Available Validators

- `brand-checklist.md`
- `ui-checklist.md`
- `ux-checklist.md`
- `accessibility-checklist.md`
- `presentation-checklist.md`
- `documentation-checklist.md`

## Usage

1. Identify the artifact type.
2. Load the matching checklist from this directory.
3. Review the artifact against each checklist item.
4. Mark each item as pass, fail, not applicable, or pending.
5. Escalate unresolved failures to the listed owner.
